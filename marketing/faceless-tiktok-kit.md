## AI Voice Settings

### ElevenLabs Recommended Settings

```
Voice: "Adam" or "Antoni" (natural male)
        "Rachel" or "Bella" (natural female)
        
Stability: 0.5 (balanced)
Clarity: 0.75 (clear but not robotic)
Style: 0 (neutral)

Speed: Normal or slightly slow (0.9x if option exists)
```

### CapCut TTS Alternative

```
Voice: "Alex" (male) or "Emma" (female)
Speed: Normal
Pitch: Default

Note: CapCut TTS is faster but less natural
ElevenLabs is better quality but requires export/import
```

### Voice Script Tips

- Use periods for pauses
- Write "point one eight" not "0.18" (clearer speech)
- Keep sentences short
- End with downward inflection (sounds confident)

## CapCut Editing Workflow

### Step 1: Import Screen Recording
- Record at 1080x1920 (vertical)
- Include full flow: connect ‚Üí scan ‚Üí results ‚Üí cleanup ‚Üí success

### Step 2: Add AI Voiceover
- Option A: Generate in ElevenLabs ‚Üí import audio
- Option B: Use CapCut's built-in TTS

### Step 3: Cut to Match Audio
- Align screen actions with voiceover beats
- Use J-cuts (audio leads video slightly)

### Step 4: Add Text Overlays
- Use CapCut "Auto Captions" for the voiceover
- Add manual overlays for key moments:
  - Hook text (0-3s) - BIG, bold
  - Stats ("47 accounts", "0.18 SOL")
  - CTA ("link in bio")

### Step 5: Add Sound Effects
- CapCut has free SFX library
- Use sparingly:
  - Notification ding (hook)
  - Cash register (when showing SOL amount)
  - Success chime (when cleanup completes)

### Step 6: Add Badges/Trust Elements
- Create in Canva or use CapCut stickers:
  - "Non-custodial"
  - "You Sign Everything"
  - "No Seed Phrases"

### Step 7: Export
- 1080x1920
- 30fps
- High quality
- Direct upload to TikTok

## Text Styles (CapCut)

### Hook Text (0-3s)
```
Font: Bold/Heavy (CapCut "Impact" or "Bebas")
Size: Large (fills 60% of screen width)
Color: White with black outline
Position: Center
Animation: Pop or Scale In
Duration: 2-3 seconds
```

### Info Text (stats, steps)
```
Font: Clean sans-serif (CapCut "Montserrat" or "Poppins")
Size: Medium
Color: White
Position: Top third or bottom third
Animation: Fade or Slide
Background: Semi-transparent black bar
```

### CTA Text
```
Font: Bold
Size: Medium-Large
Color: Green (#00FF88) or Yellow
Position: Center-bottom
Animation: Pulse or Glow
Add: Arrow pointing down (‚Üì) or "link in bio" with finger emoji
```

## Batch Production Method

### Record Once, Make 5 Videos

1. **Record 3 different wallet cleanups** (vary the amounts)
2. **Cut each into:**
   - Full 30s demo
   - Quick 15s highlight
   - Specific moment clip (just the results reveal)
3. **Remix with different:**
   - Voiceovers (male/female)
   - Hooks (swap first 3 seconds)
   - Music/sounds

### Example Batch Day

```
Morning (1 hour):
- Record 3 wallet cleanups (15 min each)
- Export raw footage

Afternoon (2 hours):
- Script A: Full demo √ó 3 wallets = 3 videos
- Script B: Quick cut √ó 3 wallets = 3 videos
```
